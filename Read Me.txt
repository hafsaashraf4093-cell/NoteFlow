# NoteFlow - Smart Notes Web Application with AI Suggestions

## Project Overview

NoteFlow is a web-based note management application developed using Python Flask, SQLite, HTML, CSS, and JavaScript. The purpose of this project is to help users create, organize, manage, and search notes efficiently while improving productivity through intelligent suggestions.

The application provides secure user authentication, note management features, analytics, settings management, responsive design, and a smart suggestion system.

# How the Project Works

## Step 1: Running the Application

The project is executed through Python Flask.

Command:

python app.py

or

python main.py

After execution, Flask generates a localhost URL such as:

http://127.0.0.1:5000

The generated URL is copied and opened in a web browser to access the application.

# Smart Suggestion System

One of the key features of NoteFlow is the intelligent suggestion system.

The system analyzes note content and generates useful recommendations based on predefined keywords.

Examples:

Keyword: "exam"
Suggestion: Study planning recommendation

Keyword: "important"
Suggestion: Pin this note

Keyword: "meeting"
Suggestion: Create action items

Currently, the project uses a rule-based intelligent suggestion engine that simulates AI assistance.

Future versions may integrate real Artificial Intelligence APIs for more advanced suggestions.
